Version 1.1

  • Now by default only new videos are shown when clicking the extension icon.
    That way you don't have to go through each account to see new uploaded videos.

Version 1.2

  • Added a cache system for new uploaded videos.
    This basically means that once new videos are found and loaded, they won't be fetched from Youtube every time you click the extension icon. This speeds up the process of watching several new videos. The cache is cleaned once new videos are found in the background.
  • Added this "History" page for those interested in what is new/changed.
  • Added a "Clean Cache" button to the options page. This should be self explanatory.
  • Updated options page with social media links and relevant information.

Version 1.2.2

  • Added the option to mark videos as watched.
    Just right-click on the video you want, and click where it says "Mark videos as watched"
